Poseidon Temperature Import

Temperature data import and storage from Poseidon 2250.

Temperature data readout from external system (xml).
Data direct storage in Odoo database.
Data review and printout.

This module: 1) Adds a security group named Temperature, so you can give access to users through Settings--> Users---> Users menu.
2) Creates Temp Readout scheduler in Settings--> Technical---> Automation---> Scheduled action. Change readout time acording your demands.
3) For graph view you select the approprite graph. In the colomns select the date and in the rows select the rooms.

Main Features:

  • Scheduled data read in from remote IP:port
  • Data storage in Odoo Database
  • Report generation
  • Warning indication

Creates Temperature menu under Reporting.
Creates new User access rights "Temperature".
